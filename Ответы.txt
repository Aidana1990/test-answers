Ответы
Какой командой можно узнать текущую директорию
pwd
В этой директории находятся базовые конфигурационные файлы
/etc
Какие атрибуты будут у файла, выполнив команду chmod 760 file

-rwxrw---- file
Нужно изменить атрибуты учётной записи. Это можно выполнить при помощи программы

usermod
Какой командой можно посмотреть загруженные модули ядра Linux?

lsmod
Узнать занятое и свободное пространство дисков можно при помощи команды

free
Как можно получить количество символов в файле

wc -c file
При помощи какой программы можно изменить параметры работающего ядра Linux

sysctl
Выбери верный ответ, который позволит создать символическую ссылку

ln -s /opt/file /tmp/system
Безопасно и корректно отредактировать sudoers нужно используя программу

visudo
